# React TypeScript ベースプロジェクト

Vite + React + TypeScript のベースプロジェクトです。

## インストール済みパッケージ

- **ress**: リセットCSS
- **react-i18next / i18next**: 多言語対応
- **react-icons**: アイコンライブラリ
- **@tanstack/react-query**: キャッシュ管理
- **react-datepicker**: カレンダー入力

## 使い方

### 開発サーバー起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

### プレビュー

```bash
npm run preview
```
