# プロジェクト説明書

## プロジェクト構成

このプロジェクトは、Vite + React + TypeScript をベースとした、すぐに使えるベースプロジェクトです。

### ディレクトリ構造

```
tel_demo/
├── sample/              # React TypeScript プロジェクト
│   ├── src/
│   │   ├── i18n/        # 多言語対応設定
│   │   ├── App.tsx      # メインコンポーネント
│   │   ├── App.css      # アプリスタイル
│   │   ├── index.css    # グローバルスタイル（CSS変数定義）
│   │   └── main.tsx     # エントリーポイント
│   └── package.json
└── プロジェクト説明書.md
```

## セットアップ

### 1. React TypeScript のプロジェクト作成

```bash
npm create vite@latest sample -- --template react-ts
cd sample
npm install
```

### 2. 必要なパッケージのインストール

#### リセットCSSの導入

```bash
npm install --save ress
```

#### 多言語対応

```bash
npm install react-i18next i18next
```

#### アイコンの導入

```bash
npm install react-icons
```

#### キャッシュ管理

```bash
npm install @tanstack/react-query
```

#### カレンダー入力

```bash
npm install react-datepicker @types/react-datepicker
```

### 3. 一括インストール

すべてのパッケージを一度にインストールする場合：

```bash
cd sample
npm install --save ress react-i18next i18next react-icons @tanstack/react-query react-datepicker @types/react-datepicker
```

## 実行方法

### ルート階層から実行（推奨）

プロジェクトのルート階層（`tel_demo`）から実行する場合：

```bash
# 開発サーバー起動
npm --prefix sample run dev

# ビルド
npm --prefix sample run build

# プレビュー
npm --prefix sample run preview

# リント
npm --prefix sample run lint
```

### sampleディレクトリ内から実行

```bash
cd sample

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview

# リント
npm run lint
```

## 主な機能

### CSS変数による一元管理

`src/index.css`の`:root`で、メインカラー・フォント・フォントサイズを一元管理しています。

**定義されている変数：**
- **メインカラー**: `--color-primary`, `--color-text`, `--color-bg` など
- **フォント**: `--font-family`
- **フォントサイズ**: `--font-size-base`, `--font-size-sm`, `--font-size-lg` など

他のCSSファイルでは、`var(--color-primary)`のように使用できます。

### 多言語対応

`src/i18n/config.ts`で設定済み。日本語・英語に対応しています。

### キャッシュ管理

`@tanstack/react-query`を使用したデータフェッチングとキャッシュ管理が設定済みです。

## 開発のヒント

- CSS変数は`src/index.css`の`:root`で一元管理されています
- 多言語対応の翻訳は`src/i18n/config.ts`で追加・編集できます
- アイコンは`react-icons`からインポートして使用できます
- カレンダー入力は`react-datepicker`を使用できます
